# html
## 结构
<(opening tag)  (attribute)> content </closing tag>\
上述称为一个element

一个html文件大概布局
```html
<!doctype html> 声明文档类型
<html lang="ah-CN">
<html>
<head>
<meta charset="utf-8" /> 不能由其他html相关元素表示的元数据

<title>xxx</title> 页面的标题
</head>

<body>
<!-- 其中为注释内容 -->

</body>
</html>
```

